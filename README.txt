For use eslint for base js use  .eslintrc from the root directory
For use ReactJS specified options use .eslintrc file from the react folder

SETUP WEBSTORM FOR USE ESLINT FILE

1) go to Setting(Preferences) -> ESLINT Settings
2) set checkbox enabled
3) select used at project node version
4) select installed eslint module (use eslint version 5 for IDE version lower when 2019.2)
5) select Use specific config file and choose needed .eslintrc file
